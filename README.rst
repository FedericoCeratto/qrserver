
qrserver 
======== 
 
A simple tool to serve a file to a smartphone using a QR code. 
 
Usage 
----- 

.. code-block:: bash
  
    Help:
    ./qrserver.py -h
    
    Serve a file:
    ./qrserver.py README.rst
